# Modern Banking Automation System 🏦

A robust, simulated desktop banking application built with Python and Tkinter, featuring secure authentication, role-based access control, and an automated SQLite transaction ledger. 

*Disclaimer: This is a simulated environment created for portfolio and practice purposes. No real-world emails or financial transactions are executed.*

## 🚀 Key Features
* **Role-Based Dashboards:** Distinct interfaces and permissions for Admin and standard User accounts.
* **Relational Database Management:** Utilizes `SQLite3` to securely handle user credentials, balances, and real-time transaction ledgers.
* **Automated Data Workflows:** Features simulated email dispatches for account creation and secure password recovery.
* **Dynamic Analytics Ledger:** Automatically calculates and logs credits, debits, and updated balances into a searchable transaction history.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **GUI Framework:** Tkinter & tkintertable
* **Database:** SQLite3
* **Asset Handling:** Pillow (PIL)
